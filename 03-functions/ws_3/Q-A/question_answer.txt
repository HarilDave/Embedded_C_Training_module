The header file required to work with variadic functions is      <stdarg.h>.

The macro  va_start  is used to initialize the variable argument list.

va_arg is used to fetch each argument from the variadic list.

The macro used to clean up the variable argument list is va_end.

Function printf is a common example of a variadic function in C.
